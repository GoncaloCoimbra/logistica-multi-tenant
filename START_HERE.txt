┌────────────────────────────────────────────────────────────────────────────┐
│ 📋 ENDPOINTS PRINCIPAIS                                                    │
└────────────────────────────────────────────────────────────────────────────┘

    HISTÓRICO:
    ├─ GET  /api/audit-logs              → Listar todos
    ├─ GET  /api/audit-logs/entity/:...  → Por entidade
    ├─ GET  /api/audit-logs/user/:...    → Por usuário
    └─ GET  /api/audit-logs/stats        → Estatísticas

    NOTIFICAÇÕES:
    ├─ GET  /api/notifications           → Listar todas
    ├─ GET  /api/notifications/unread    → Não lidas
    ├─ GET  /api/notifications/unread/count
    ├─ PATCH /api/notifications/:id/read → Marcar como lida
    ├─ PATCH /api/notifications/read-all → Todas como lidas
    └─ DELETE /api/notifications/:id     → Deletar

    PRODUTOS (Dispara notificações):
    ├─ POST /api/products                → Cria + notificação
    └─ PUT  /api/products/:id/status     → Atualiza + notificação
