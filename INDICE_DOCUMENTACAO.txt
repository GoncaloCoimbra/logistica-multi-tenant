1. Recompile o backend:
   cd backend-nest
   npm install
   npm run start:dev

2. Depois de compilar, escolha uma opção:

   OPÇÃO A (Manual):
   - Abra GUIA_TESTE_MANUAL.md
   - Siga os passos com Postman

   OPÇÃO B (Automatizado):
   - Execute: .\test-audit-notifications.ps1

   OPÇÃO C (BD Direto):
   - Use SQL queries em GUIA_TESTE_MANUAL.md

3. Valide o resultado:
   - Histórico criado? 
   - Notificação criada?
